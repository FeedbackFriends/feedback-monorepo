# OpenAPI Contract

`feedback-api.yaml` is the canonical OpenAPI contract for the backend.

- Source of truth: backend Spring controllers, DTOs, and OpenAPI config
- API contract version: stable `info.version` managed in backend OpenAPI config and changed only for deliberate API-version changes
- Generated by: `cd backend && SPRING_PROFILES_ACTIVE=openapi ./gradlew syncOpenApiSpec --no-configuration-cache`
- Regenerate all committed artifacts from the repo root with: `./scripts/generate-api-artifacts.sh`
- Consumed by:
  - frontend type generation in `web/src/lib/api/generated/openapi.ts`
  - iOS Swift OpenAPI generation in `ios-app/Modules/Sources/OpenAPI/GeneratedSources/`

Do not edit `feedback-api.yaml` by hand. Regenerate it from the backend whenever the API contract changes.
